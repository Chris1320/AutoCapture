# AutoCapture

Simplify picture-taking and keeping track of your subjects' details.

**How It Works**:

1. Add a new entry for a subject by clicking the "_Add_" button.
2. Fill in the subject's details, including name, age, and any other relevant information.
3. Use the camera feature to take a picture of the subject. Retake the picture if necessary until you are satisfied with the result.
4. Save the entry and view the subject's information and picture at any time.
5. Use the "_Edit_" button to update the subject's details or change the picture if needed.
6. When finished for all subjects, click "_Done_" to generate an Excel spreadsheet containing all the collected information and picture filepaths.
